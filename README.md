# 🐘 PostgreSQL — 12 учебных проектов (Школа 21)

## 📚 О репозитории

Этот репозиторий содержит **12 учебных проектов** по PostgreSQL, выполненных в рамках обучения в **Школе 21** (проект Сбера).

Каждый проект — это отдельная тема, которая включает в себя от 2 до 13 SQL-заданий. Всего в репозитории **90 заданий**, покрывающих практически все аспекты работы с PostgreSQL — от простых SELECT-запросов до построения ETL-процессов и работы с темпоральными данными.

---
