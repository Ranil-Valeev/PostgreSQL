# SQL Project 12 — Data Warehouse

## 📚 О проекте

Это **групповой проект**, в котором я разобрался с устройством хранилищ данных (DWH) и прошел весь путь создания ETL-процесса.

В проекте я научился:
- Создавать SQL-запросы для обработки «неидеальных» данных
- Искать пропущенные значения и связывать информацию из разных источников
- Корректно агрегировать показатели с учетом истории изменений (темпоральные данные)
- Рассчитывать курсы валют по ближайшей доступной дате

Эти навыки необходимы для построения ETL-пайплайнов, анализа данных с историей изменений и подготовки данных для отчетности.
