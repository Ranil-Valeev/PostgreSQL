# SQL Project 9 — Aggregated data is more informative, isn't it?

## 📚 О проекте

В этом проекте я освоил ключевые навыки SQL-анализа для решения реальных бизнес-задач:

- Агрегацию данных (расчет сумм, средних, минимума/максимума)
- Группировку по атрибутам (GROUP BY)
- Фильтрацию результатов (HAVING)
- Соединение таблиц (JOIN)
- Работу с условиями и преобразование типов

Я научился превращать сырые данные в структурированные отчеты и рассчитывать метрики эффективности.
