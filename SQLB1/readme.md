# SQL Project 1 — Реляционная модель и SQL

## 📚 О проекте

В этом проекте я научился:
- Создавать SQL-запросы с использованием внутренних подзапросов в частях `FROM` и `SELECT`
- Фильтровать данные по диапазону дат
- Работать с условиями выборки и сортировкой по нескольким полям
- Структурировать запросы для получения нужной информации
