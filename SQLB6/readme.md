# SQL Project 6 — Traveling Salesman Problem (TSP)

## 📚 О проекте

В этом **групповом проекте** я решил классическую задачу коммивояжёра (Traveling Salesman Problem) с использованием SQL.

Я освоил:
- Моделирование графов и маршрутов в SQL
- Создание таблиц с узлами и стоимостью перемещений
- Написание рекурсивных запросов для поиска всех возможных маршрутов
- Поиск оптимальных путей с минимальными и максимальными затратами
- Анализ результатов с сортировкой

---

## 📋 Список заданий

| № | Название | Описание | Статус |
|---|----------|----------|--------|

| **01** | Opposite TSP | Добавить маршруты с максимальной стоимостью к результату | ✅ |
